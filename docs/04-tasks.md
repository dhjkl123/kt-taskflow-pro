## 진행 규칙

### ✅ 필수 규칙

1. **순서대로만 진행**
   - Phase 1 완료 → Phase 2 시작
   - 각 Phase 내 단계도 순서 유지
   - 병렬 작업 금지

2. **단계별 검증 필수**
   - 각 단계의 "검증 방법" 실행 필수
   - 검증 실패 → 현재 단계에서 멈춤
   - 검증 통과 후 다음 단계 진행

3. **상태 추적**
   - ✅ 완료: 검증 통과
   - ⏳ 진행 중: 현재 작업 중
   - ⏳ 대기: 이전 단계 완료 대기

4. **문제 발생 시**
   - 현재 Phase에서 멈춤
   - CLAUDE.md 상담 (절대 규칙 적용)
   - 진행 계획 수정 후 재시작
