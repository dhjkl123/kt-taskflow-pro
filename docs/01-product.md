# 01. 제품 정의

## 프로젝트 목표

**팀 업무 시각화로 '지금 누가 뭐해?'라는 물음이 사라지게 하기**

스타트업은 자주 진행 상황을 재확인합니다. 팀원이 뭐 하고 있는지, 마감은 언제인지 물어봅니다. TaskFlow Pro는 모든 업무와 진행 상황을 한 화면에 시각화하여 불필요한 커뮤니케이션을 줄입니다.

---

## 타겟 페르소나

| 항목 | 설명 |
|------|------|
| **규모** | 10명 규모 스타트업 |
| **역할** | 팀 리더 (CTO, PO, PM) |
| **나이대** | 30~40대 |
| **주요 니즈** | 팀 업무 상황을 빠르게 파악, 마감 관리 |
| **사용 환경** | 사무실(데스크톱), 외출 중(모바일) |

---

## MVP 범위

### 핵심 기능

#### 1. CRUD 4종 (모두 화면에서 수행 가능)
- ✅ **Create**: 업무 추가
- ✅ **Read**: 업무 목록 조회
- ✅ **Update**: 업무 수정
- ✅ **Delete**: 업무 삭제

#### 2. 업무 상태 분류
- `To Do` - 대기 중
- `In Progress` - 진행 중
- `In Review` - 검토 중
- `Done` - 완료

#### 3. 마감 시각 (due_at)
- **형식**: 날짜 + 시간 (예: `2026-05-12 18:00`)
- **표시**: 마감까지 남은 시간 상대 표시 (예: "3시간 후", "어제")
- **색상 강조**: 긴급(빨강), 주의(주황), 정상(회색)

#### 4. 테마 토글
- **라이트 / 다크 모드** 선택 가능
- **저장소**: localStorage에 사용자 선택 저장
- **기본값**: 시스템 설정 따르기

#### 5. 모바일 반응형
- **최소 너비**: 360px (삼성 Galaxy S5)
- **지원**: 데스크톱, 태블릿, 모바일 모두
- **터치 친화적** UI

## UI/UX 방향

### 디자인 언어: macOS 스타일

```
특징:
- 둥근 모서리 (8px ~ 12px border-radius)
- 부드러운 그림자 (subtle box-shadow)
- 반투명 카드 배경 (glass-morphism, 80-90% opacity)
- 시스템 폰트 사용 (-apple-system, BlinkMacSystemFont)
- 공간을 활용한 미니멀한 디자인
- 깔끔한 색상 (백/흑 + 1가지 accent color)
```

### 색상 팔레트 (예시)
- **Primary**: 파란색 (업무 선택, 강조)
- **Success**: 초록색 (완료)
- **Warning**: 주황색 (마감 24시간 이내)
- **Danger**: 빨강색 (마감 초과)
- **Background**: 흰색 / 다크 그레이 (라이트/다크)

## 범위 외 (Out of Scope)

### MVP에서 제외

| 항목 | 이유 |
|------|------|
| **외부 결제** | MVP에는 불필요 |
| **네이티브 앱** (iOS/Android) | 웹 앱으로 충분 |
| **WebRTC** (화상회의) | 슬랙 같은 도구와 중복 |

| **WebSocket** (실시간) | **MVP는 폴링 사용, 확장 단계에서 검토** |

## 성공 기준 (Definition of Done)

### 기술적 성공 기준

- ✅ **새로고침 후에도 데이터 유지**
  - localStorage 또는 서버 DB에서 복구
  
- ✅ **360px 화면에서 UI 깨짐 없음**
  - 모바일 테스트 필수
  
- ✅ **API 응답 시간 < 200ms**
  - 네트워크 지연 제외
  
- ✅ **CRUD 4종 모두 화면에서 동작**
  - 추가: 입력 폼 필수
  - 목록: 테이블 또는 카드 리스트
  - 수정: 인라인 또는 모달 수정 폼
  - 삭제: 확인 다이얼로그 필수
  
- ✅ **테마 토글 작동**
  - 버튼 클릭 시 즉시 전환
  - localStorage에 저장
  - 새로고침 시 설정 유지

### 사용자 만족도

- 팀 리더가 5초 안에 전체 업무 상황 파악 가능
- 마감 임박 업무가 눈에 띄게 표시
- 모바일에서도 기본 기능 사용 가능
